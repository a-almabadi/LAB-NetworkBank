# Bank Network Project

This project involves planning and developing a **secure bank network** using **Cisco Packet Tracer**.  
Multiple networks were connected together to ensure smooth communication between branches and the main bank using protocols such as **SSH**, **AAA**, and **NAT**.

---

## Tools Used
- Cisco Packet Tracer  
- Networking Protocols (SSH, AAA, NAT)  
- Basic Network Security Concepts
